# Model Card Template

## Intended Use

Credit risk exploration and fairness education using public or synthetic data.

## Data

Describe source, features, protected groups, target, and known limitations.

## Metrics

Include precision, recall, F1, accuracy, calibration error, statistical parity difference, equal opportunity difference, and equalized odds difference.

## Ethical Considerations

Document known risks, subgroup performance gaps, and mitigation strategy.

## Limitations

This project is not a production credit approval system.
